### Bootstrap $\hat{r}^2$

``` r
bootstrap_r2 = 
  weather_df %>%  
  modelr::bootstrap(n=5000) %>%  
  mutate (
    models = map(strap, \(df) lm(tmax ~ tmin + prcp, data=df)),
    results = map(models,  broom::glance) 
  ) %>%  
  select(-strap, -models) %>%  
  unnest(results) %>%  
  select(r.squared)
```

``` r
bootstrap_r2 %>% 
  ggplot(aes(x = r.squared)) + geom_density()
```

<img src="p8105_hw6_sc5126_files/figure-gfm/plot_r2-1.png" width="90%" />

Here the $\hat{r}^2$ value we derived is high. It has a mean of
0.9170951. the bootstrap values of $\hat{r}^2$ ranges from 0.8699127 to
0.9541121. It can be concluded that `tmin` and `prcp` explained `tmax`
decently.

### Bootstrap $\log(\beta_1 * \beta2)$

``` r
bootstrap_logb1b2 = 
  weather_df %>% 
  modelr::bootstrap(n = 5000) %>% 
  mutate(
    models = map(strap, \(df) lm(tmax ~ tmin + prcp, data=df)),
    results = map(models, broom::tidy)
    ) %>% 
  select(-strap, -models) %>% 
  unnest(results) %>% 
  select(id = `.id`, term, estimate) %>% 
  pivot_wider(
    names_from = term, 
    values_from = estimate
    ) %>% 
  rename(
    beta0 = `(Intercept)`, 
    beta1 = tmin,
    beta2 = prcp
    ) %>% 
  mutate(
    log_b1b2 = log(beta1 * beta2)
    )
```

``` r
 bootstrap_logb1b2 %>% 
  ggplot(aes(x = log_b1b2)) + geom_density()
```
